開發時的技術心得

theme-toggle

  • HTML: index.html 檔

    HTML檔 先定義好 SVG 元素區塊的範圍，SVG 中共含有三部份主要內容，分別是：mask 元素、太陽圓形主體、太陽光芒。 mask 元素中創造一個白色矩形跟黑色圓形，mask 用來遮擋部份的太陽圓形主體，讓圖形變成彎彎的月亮，如此可讓太陽跟月亮在同一個 SVG 內部来實現。

  • CSS: theme.css 檔

    theme.css 分別定義了白天模式和夜間模式下的視覺風格，包括文字顏色、背景顏色、圖標填充顏色以及動畫效果。藉由 HTML 中根元素的 data-theme 屬性被設置為 light 或 dark，決定相對應的 CSS 變量，以改變網頁的外觀。另外，在主題切換時做了一些動畫和過渡效果，像是：太陽主體的縮放變化（transform: scale()）、太陽光芒的不透明度變化（opacity）、月亮的位置變化（transform: translateX()）。

  • Javascripts: theme.js 檔

    當使用者打開網頁時，theme.js 會調用 reflectPreference 函式，該函式會再調用 getColorPreference 函式， 其先檢查 localStorage 中，是否有使用者上一次使用網頁時選擇的模式（選擇會被記住，因此即便關閉瀏覽器再重新打開， 網站仍然會顯示上一次選擇的主題模式），若無，則依當前系統模式返回，讓 reflectPreference 函式將網頁 html 元素的 data-theme 屬性設置為其返回值。網頁載入完成後，在切換按鈕上設置監聽器，監聽點擊事件。當使用者點擊按鈕時，會觸發 onClick 函式，進行日夜模式切換，然後調用 setPreference 函式保存使用者的選擇，並調用 reflectPreference 函式更新網頁上的主題模式顯示。
